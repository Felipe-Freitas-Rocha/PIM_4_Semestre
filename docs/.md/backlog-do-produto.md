## :package: Backlog do Produto
- As seguintes funcionalidades foram priorizadas e implementadas para a entrega do projeto:

[x] Registro de solicitações de suporte: Colaboradores podem criar novos chamados detalhando seus problemas.
[x] Sugestões automáticas via IA: O sistema analisa o chamado e sugere uma prioridade.
[x] Encaminhamento ao técnico responsável: A IA atribui o chamado ao técnico especialista mais adequado.
[x] Priorização de chamados: A IA define a urgência e o técnico pode confirmá-la.
[x] Relatórios gerenciais: Endpoint que fornece estatísticas gerais do sistema para Administradores.
[x] Histórico de solicitações por usuário: Cada usuário (Colaborador e Técnico) tem acesso ao seu histórico de chamados.
[ ] Integração com notificações (pop-up): Funcionalidade planejada para versões futuras.
